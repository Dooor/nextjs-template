# Next.js Template Project

This is a Next.js template project with TypeScript, React, Redux, Immutablejs and Express.

## Packages

- Typescript
- React
- Redux
- Immutablejs
- Express

## How to use it?

### Download

```bash
git clone git@github.com:Dooor/nextjs-template.git
cd nextjs-template
```

### Install it

```bash
npm install
```

### Run dev server

```bash
npm run dev
```

### Build

```
npm run build
```

### Test

```
npm run test
```

### Check type

```
npm run type-check
```

### Check lint

```
npm run link-check
```
